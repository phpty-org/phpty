# PhPty

## Agent skills

### Issue tracker

Issues are tracked on GitHub (phpty-org/phpty) via the `gh` CLI; external PRs are also a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: `CONTEXT-MAP.md` at the root points to per-context `CONTEXT.md` files (`vterm/`, `yamatanooroti/`). See `docs/agents/domain.md`.
